Bittypak은 Windows 운영체제에서 실행될 수 있는 오디오 플레이어입니다.
간단한 기능을 포함하고 있으며 DRAG & DROP을 지원합니다.

시스템과 관련된 기능을 일부 사용하고 있으며 사용자가 작업한 내용을 기록하기 위한 용도로 활용됩니다.
예를 들어, 마지막 위치나 사용자가 불러온 미디어 파일을 기억해두었다가 프로그램이 시작될 때 이 목록을 탐색하여 필요한 내용을 불러옵니다.

이외에 별 다른 기능은 없으며 단순히 오디오 파일을 재생하는 기능만을 담고 있습니다.
일부 제한된 영역에서 스트레오 믹스, 즉 기본 스피커로 출력되는 사운드를 기록할 수 있습니다.

이 프로그램의 소스 코드는 공개하되 빌드 스크립트는 공개하지 않으며,
기본적으로 Git과 Github의 정책에 따라 저작권이 보호됨을 알립니다.
